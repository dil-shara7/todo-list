# Todo-list System - User Documentation

## Project Overview
TaskFlow is a web-based task management app that allows users to:
- Create tasks with priorities, due dates, notes, attachments, and links
- Track task progress
- Set reminders for upcoming or overdue tasks
- Filter tasks by status, priority, and due date
- Edit and delete tasks easily

## **Frontend Developer Role**
Responsibilities include:
- UI/UX implementation
- Frontend feature development
- Integration with backend
- README.md creation and maintenance
- User documentation

## **Technologies Used**
- HTML, CSS, JavaScript 
- LocalStorage for data persistence
- Google Fonts for typography
- Responsive design with CSS

# User Guide 

##  Logging In
1. Open the app.
2. Enter your email and password.
3. Click **Sign In**.
4. You will be redirected to the main app dashboard.

##  Creating a Task
1. Click **Add New Task**.
2. Fill in the task title (required).
3. Optionally add:
   - Notes or description
   - Priority (Low, Medium, High)
   - Due date & time
   - Links and attachments
   - Reminder
   - Progress (0-100%)
4. Click **Add Task*

##  Managing Tasks
- **Edit Task:** Click the green **Edit** button, make changes, and save.
- **Delete Task:** Click the red **Delete** button.
- **Mark Complete:** Click the checkbox next to the task title.

##  Filtering Tasks
Use the sidebar to filter tasks by:
- **Status:** All, Active, Completed
- **Priority:** Low, Medium, High
- **Due Date:** Today, This Week, Overdue

##  Notifications
- Task reminders will appear as popups in the browser.
- Overdue tasks also trigger notifications.

##  Tips
- Task data is stored in your browser.
- Refreshing the page will retain tasks.
- Enable notifications for timely reminders.
